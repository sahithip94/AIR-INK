# 🎨 Air Ink

Air Ink is a browser-based virtual drawing application that allows users to draw in the air using hand gestures and a webcam. Built with React and MediaPipe Hands, it provides a touch-free drawing experience where users can sketch, erase, switch colors, and clear the canvas using intuitive gestures.

---

## ✨ Features

- 🖐️ Real-time hand tracking
- ✍️ Draw using your index finger
- 🎨 Multiple brush colors
- 🧽 Eraser tool
- 🗑️ Clear the entire canvas
- 📷 Webcam-based interaction
- ⚡ Smooth and responsive drawing experience
- 💻 Clean and interactive user interface

---

## Getting Started

### Clone the repository

```bash
git clone https://github.com/sahithip94/AIR-INK.git
```

### Navigate into the project

```bash
cd AIR-INK
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```

---

##  How to Use

1. Launch the application.
2. Allow access to your webcam.
3. Use your index finger to draw in the air.
4. Select different colors from the color palette.
5. Switch to the eraser when needed.
6. Clear the canvas using the designated gesture.
